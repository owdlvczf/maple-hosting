## The Network: Actually Impressive

One thing that stands out is the network architecture. Maple-Hosting runs on a blend of premium transit providers — Telia, Cogent, NTT, Liberty Global — plus peering via AMS-IX. On top of that, they use the **Noction Intelligent Routing Platform**, which automatically reroutes packets around congestion and packet loss in real time. According to their documentation, this can improve delivery performance by up to 50%.

They connect to six backbone providers via 100GE uplinks and use Juniper routers with redundant routing engines, redundant fiber rings, and A+B power feeds. It's the kind of network setup you'd expect at a much larger hosting provider.
